# を研究所

「を研究所」は、日本語の「を」の使い方を研究、発掘するためのWebアプリケーションです。

## 概要
SQLite (WebAssembly) を利用し、ブラウザ内で動作するデータベース駆動型のツールです。名詞・動詞の組み合わせや例文の探索、自分だけのお気に入りリスト作成が可能です。

## 使用方法

### モード共通

| キー | 操作 |
| :--- | :--- |
| **A / D** | モード切り替え（左回り / 右回り） |
| **W / S** | リスト項目の選択 (上 / 下) |
| **SPACE** | 例文またはリストの更新 |
| **R** | 登録モード（直接入力）の起動 |

### 例文モード
WIKIPEDIA全文から**（名詞）を（動詞）**にマッチした例文がランダムで表示されます。
| キー | 操作 |
| :--- | :--- |
| **Q** | 例文を検索する|
| **F** | 名詞を保存 |
| **SHIFT + F** | 動詞を保存 |

### 名詞モード 
保存した名詞が表示されます。
| キー | 操作 |
| :--- | :--- |
| **Q** | 名詞を検索する|
| **F** | 保存または削除（既に保存されている場合は削除） |

### 動詞モード 
保存した動詞が表示されます。
| キー | 操作 |
| :--- | :--- |
| **Q** | 動詞を検索する|
| **F** | 保存または削除（既に保存されている場合は削除） |

### 作文モード
名詞と動詞を組み合わせて新しい文を作成します。
| キー | 操作 |
| :--- | :--- |
| **Z** | 例文の名詞と動詞をランダムに組み合わせて作文|
| **X** | 保存した名詞と動詞をランダムに組み合わせて作文 |
| **C** | 保存した選択中の名詞と全ての動詞を組み合わせて作文 |
| **V** | 保存した選択中の動詞と全ての名詞を組み合わせて作文 |
| **F** | 作った文を保存|

### 名文モード
保存した文が表示されます。
| キー | 操作 |
| :--- | :--- |
| **F** | 保存または削除（既に保存されている場合は削除） |

### 検索モード
ワードを入力して検索ができます。例文モード、名詞モード、動詞モードのみ可能です。
| キー | 操作 |
| :--- | :--- |
| **Enter** | 入力項目の切り替え |
| **F** | 検索実行 |
| **ESC** | 検索終了 |

### 登録モード
ワードを直接入力して保存ができます。名詞のみ入力されている場合は名詞を保存、動詞のみ入力されている場合は動詞を保存、名詞と動詞が両方入力されている場合は名文に保存されます。
| キー | 操作 |
| :--- | :--- |
| **Enter** | 入力項目の切り替え |
| **F** | 登録実行 |
| **ESC** | 登録終了 |


## 開発・技術スタック
- **データベース**: SQLite (via `@sqlite.org/sqlite-wasm`)
- **ストレージ**: OPFS (Origin Private File System)
- **UI**: Vanilla JavaScript / CSS
